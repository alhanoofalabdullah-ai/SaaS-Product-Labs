# 🚀 Project 06 – SaaS Project Management & Task Collaboration System

## Overview

This project demonstrates a SaaS Project Management and Task Collaboration System built using Java.

Project management platforms are widely used by organizations to plan work, assign tasks, track progress, manage deadlines, collaborate across teams, and monitor delivery performance.

This project simulates a SaaS platform similar to Asana, Monday.com, ClickUp, Jira, and Trello.

---

# Business Scenario

A company manages multiple internal and client projects.

Teams need a centralized platform to:

- Create projects
- Add tasks
- Assign team members
- Track task status
- Monitor priority
- Review overdue work
- Generate project progress reports

---

# Objectives

- Manage projects
