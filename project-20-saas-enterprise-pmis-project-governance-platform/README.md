# 🚀 Project 20 – SaaS Enterprise PMIS & Project Governance Platform

## Overview

Enterprise PMIS & Project Governance Platform is a SaaS solution designed to manage projects, governance processes, risks, issues, milestones, workflows, approvals, and project documentation within a centralized environment.

The platform simulates enterprise systems similar to:

- Oracle Aconex
- PMWeb
- Primavera Unifier
- Autodesk Construction Cloud
- Oracle Primavera P6 EPPM
- Microsoft Project Online

This project focuses on governance, project controls, documentation management, workflow automation, and executive reporting.

---

# Business Scenario

A large organization manages multiple strategic projects across different business units.

Project Management Office (PMO) requires a centralized platform to:

- Manage project portfolios
- Monitor milestones
- Track project risks
- Track issues and actions
- Manage workflows
- Govern project documentation
- Generate executive dashboards
