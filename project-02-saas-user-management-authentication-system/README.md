# 🚀 Project 02 – SaaS User Management & Authentication System

## Overview

This project demonstrates a SaaS user management and authentication system using Java.

The system simulates core SaaS account operations such as user registration, login validation, role management, account activation, and authentication workflows.

---

## Business Scenario

A SaaS platform needs a secure user management module that allows customers and administrators to access the system based on their roles.

---

## Features

- User Registration
- Login Authentication
- Password Hashing Simulation
- Role-Based Access
- Account Status Management
- User Listing
- Authentication Controller Layer
