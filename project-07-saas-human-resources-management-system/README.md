# 🚀 Project 07 – SaaS Human Resources Management System (HRMS)

## Overview

This project demonstrates a SaaS Human Resources Management System (HRMS) developed using Java.

Human Resources Management Systems are among the most widely used enterprise SaaS platforms because organizations need centralized solutions to manage employees, attendance, leave requests, payroll, departments, and workforce reporting.

This project simulates a modern HR platform similar to Oracle HCM, Workday, SAP SuccessFactors, BambooHR, and Zoho People.

---

# Business Scenario

A growing organization employs hundreds of employees across multiple departments.

The HR team requires a centralized SaaS platform to manage:

- Employee Records
- Department Structures
- Attendance Tracking
- Leave Requests
- Payroll Processing
- Workforce Reporting

The objective is to improve workforce visibility and automate HR operations.
