# 🚀 Project 08 – SaaS Inventory Management System

## Overview

This project demonstrates a SaaS Inventory Management System developed using Java.

Inventory systems are essential for companies that manage products, stock levels, suppliers, warehouses, purchase requests, and operational availability.

This project simulates a SaaS platform similar to Zoho Inventory, Oracle NetSuite, Odoo Inventory, and SAP Business One.

---

# Business Scenario

A company manages multiple products and needs to track:

- Product records
- Stock quantity
- Low stock alerts
- Supplier details
- Stock-in transactions
- Stock-out transactions
- Inventory reports

The objective is to improve stock visibility and reduce operational shortages.

---

# Objectives

- Add products
