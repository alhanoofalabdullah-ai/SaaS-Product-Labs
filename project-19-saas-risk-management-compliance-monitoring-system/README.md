# 🚀 Project 19 – SaaS Risk Management & Compliance Monitoring System

## Overview

This project demonstrates a SaaS Risk Management and Compliance Monitoring System built using Java.

Risk and compliance platforms help organizations identify risks, monitor controls, track audit findings, measure compliance status, and generate governance reports.

This project simulates enterprise GRC platforms such as ServiceNow GRC, RSA Archer, MetricStream, LogicGate, and AuditBoard.

---

# Business Scenario

An organization needs a centralized platform to monitor:

- Enterprise Risks
- Compliance Controls
- Audit Findings
- Risk Severity
- Control Effectiveness
- Compliance Status
- Governance Reports

The objective is to improve risk visibility and support executive decision-making.

---

# Objectives

- Register risks
- Classify risk severity
- Create compliance controls
- Track compliance checks
- Record audit findings
- Calculate risk exposure
- Generate governance dashboard

---

# Core Features

## Risk Management

- Create risks
- Assign severity
- Track risk status
- Monitor high-risk items

## Control Management

- Create controls
- Track control effectiveness
- Link controls to compliance activities

## Compliance Monitoring

- Record compliance checks
- Track passed and failed checks
- Calculate compliance status

## Audit Management

- Record audit findings
- Track open findings
- Support remediation monitoring

## Reporting

- Total risks
- High risks
- Controls
- Compliance checks
- Audit findings
