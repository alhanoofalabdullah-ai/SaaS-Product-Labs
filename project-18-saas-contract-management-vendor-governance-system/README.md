# 🚀 Project 18 – SaaS Contract Management & Vendor Governance System

## Overview

This project demonstrates an Enterprise Contract Management and Vendor Governance Platform developed using Java.

Organizations use contract management systems to manage vendor contracts, monitor expiration dates, track compliance obligations, assess contract risks, automate renewals, and improve governance.

This solution simulates enterprise platforms similar to:

- SAP Ariba Contracts
- Oracle Procurement Contracts
- Icertis Contract Intelligence
- Coupa Contract Lifecycle Management
- DocuSign CLM

---

# Business Scenario

A large organization manages hundreds of active contracts with suppliers, service providers, consultants, and technology vendors.

The organization requires a centralized platform to:

- Store contracts
- Manage vendors
- Track contract validity
- Monitor compliance
- Assess contract risks
- Manage renewals
- Generate governance reports
