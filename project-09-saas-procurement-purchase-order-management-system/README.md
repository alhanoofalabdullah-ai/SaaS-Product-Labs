# 🚀 Project 09 – SaaS Procurement & Purchase Order Management System

## Overview

This project demonstrates a SaaS Procurement and Purchase Order Management System built using Java.

Procurement systems are critical for organizations that need structured purchasing processes, supplier management, approval workflows, budget control, and purchase order tracking.

This project simulates an enterprise procurement platform similar to Oracle Procurement Cloud, SAP Ariba, Coupa, and Jaggaer.

---

# Business Scenario

A large organization receives procurement requests from multiple departments.

The procurement team must:

- Manage suppliers
- Review purchase requests
- Approve procurement requests
- Generate purchase orders
- Track purchasing activities
- Monitor procurement spending
- Produce procurement reports

The objective is to improve procurement governance and purchasing efficiency.
